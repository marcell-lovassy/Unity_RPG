# Unity_RPG
Point and Click RPG game made with Unity

## Features so far:

### Fighting
  1. health
  2. death
  3. animations
  
### Enemy AI
  - patroling
  - suspicion
  3. attacking
  4. chasing player

### Saving system
  1. save player and enemy positions
  2. save current scene, load it on play
  3. save health and other stats of characters and preserve data between scenes
