# Unity_RPG
Point and Click RPG game made with Unity

## Features so far:

### Fighting
  - health
  - death
  - animations
  
### Enemy AI
  - patroling
  - suspicion
  - attacking
  - chasing player

### Saving system
  - save player and enemy positions
  - save current scene, load it on play
  - save health and other stats of characters and preserve data between scenes
